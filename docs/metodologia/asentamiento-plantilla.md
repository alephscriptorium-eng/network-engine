# Plantilla — Asentamiento de sesión Aleph

Copiar, rellenar y emitir al **inicio** de cada respuesta en Modo Aleph.
El usuario puede pegar este bloque en reglas de proyecto o en el siguiente turno.

```markdown
<!-- ASENTAMIENTO_ALEPH -->
## Eigenstate del modelo (esta sesión)

- **Modelo:** [ej. Composer / Claude / otro]
- **Polo tentado:** [ej. popperiano-liberal | diamat-leninista | enciclopedista | …]
- **Sesgos estructurales activos:**
  - [inglés / tech anglosajona / …]
  - [sub-representación de …]
- **Sesgo del turno:** [qué respuesta «cómoda» me sale primero]
- **Herramientas:** [web / logs-aleph / linea-aleph / ninguna]
- **Semilla del usuario:** [cita breve]

## Tablero

- **No colapsar a:** [el polo que debo evitar hoy]
- **Superponer filtros:** [Popper | Kuhn | Gaia | diamat | … los que apliquen]
- **AutoRevisor:** [pendiente → completar tras checklist]

<!-- /ASENTAMIENTO_ALEPH -->
```

Tras el checklist en [autorevisor.md](autorevisor.md), actualizar la línea `AutoRevisor:` con el veredicto.
