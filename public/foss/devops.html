Instalación

git clone https://github.com/alephscriptorium-eng/network-engine.git
cd network-engine
pip install -e ".[dev]"

Dependencia de build

La optimización de logos y el favicon requieren Pillow. Instálala antes de generar el sitio:

pip install -e ".[build]"

Comandos CLI

nengine loadout validate default-tablero
nengine loadout apply default-tablero --semilla "tema"
nengine catalog sync
nengine build --target all
nengine session publish  # stub
nengine pack             # stub

Publicar sitio

nengine catalog sync
nengine build --target all

GitHub Actions (.github/workflows/pages.yml) sube public/ en push a main.
