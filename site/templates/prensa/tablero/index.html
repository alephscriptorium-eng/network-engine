Tablero Aleph — reglas del juego

Superposición de miradas sin colapsar. Un turno: equipar loadout → ASENTAMIENTO → cotas → forces → AutoRevisor → constelación.

Flujo del turno

  1. Equipar — skill + perfil + boot main-engine (nengine loadout apply)
  2. ASENTAMIENTO — polo tentado, sesgos, semilla del turno
  3. Cotas — ancla sima + ancla cima → posicion-linea.json (0.0–1.0)
  4. Forces Cohen — main siempre ON; ≤2 forces; 1 escena ancla cada una
  5. AutoRevisor — checklist A–H; fallo → volver a ASENTAMIENTO
  6. Tablero — 3 Alephs; constelación de miradas; salida ∅ (no cierre lineal)
